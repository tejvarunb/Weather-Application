# Weather-Application

This is a  weather application project which can we use to fethch realtime or current weather data from site.
It gives current data then follwed by the susequent five day data in one go.
The technologies used here are css,html and tailwind and javascript.
and also external API from OpenWeather is used.
I  used tailwind with CDN in HTML file itself so i havent used any node-modules.

//how to use//

1.It pretty simple and starightforwatd to use,when the site loads just type in the location of the weather you want to view and click search.
2.It give the real time current data followed by the upcoming 5 days weather report data that can be viewed.
3.By default the temperature is set to celcius we can change it to fareheit by just clicking on it.
4.and also its petty responsive across all the platforms like mobile,tablet and pc.
